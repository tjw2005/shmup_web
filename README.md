# Gradius Web-Clone

A 2D horizontal scrolling shooter inspired by Gradius, built with HTML5 Canvas and Vanilla JavaScript.

## Setup
1. Open `index.html` in any modern web browser.
2. No installation required.

## Controls
- **Arrow Keys**: Move the ship.
- **Z**: Shoot (Upgrade: Double, Laser).
- **X**: Activate selected Power-Up.

## Power-Up System
Collect **Red Capsules** to advance the selection bar at the bottom.
Press **X** to activate the highlighted upgrade:
1. **Speed Up**: Increases ship speed (Stackable).
2. **Missile**: Fires missiles that travel along the ground.
3. **Double**: Fires a second bullet upward at 45 degrees.
4. **Laser**: Fires a powerful piercing beam.
5. **Option**: (Not fully implemented) Placeholder for follower logic.
6. **Shield**: Protects against 3 hits/terrain collisions.

## Gameplay
- Destroy waves of enemies to survive and score points.
- Avoid colliding with enemies or terrain (walls/ceiling).
- Collect powerups dropped by special enemies (currently dropped randomly for testing).
